- Box 61: Picture System (searched all Fd)
  - Fd 1–2: PS-2 Product Brief (1977)
    - "Picture System 2 Product Brief", May 1977

      The available Graphics Software Packages for Picture System 2 supported
      PDP-11 operating systems DOS/BATCH, RT-11, and RSX-11M. However, an "Any
      Computer" interface is described that could use all the features of the
      PDP-11.

      Selected pages are transcribed in 1977-05_ps2_brief/.
  - Fd 7–9: Data Base Manipulation and Display System, User's Manual (1979)
    - "Data Base Manipulation and Display System User's Manual", First Edition,
      January 1979

      The Data Base Manipulation and Display System runs on a PDP-11 with
      RSX-11M.
  - Fd 10: PS-2, Unix User's Guide (1978)
    - Describes a Picture System II device driver for UNIX, which improves upon
      an earlier version which also used UNIX.

      Selected pages are transcribed in 1978-10_unix_ps2_guide/.
  - Fd 17: PS-300, Recosting Report (1985)
    - Shows costs and time to produce each product, June 12, 1985.
  - Fd 18: PS-300, Design Review (1982)
    - "Design Review for the Enhanced Graphics Control Processor (EGCP) Card for
      the PS300" \

- Box 98: Affiliated Companies and Customers (searched Fd 2—3, 7, 19, 23–24, 34)
  - Fd 2–3: Bell Northern, Correspondence and affiliation analysis (1974-1978)
    - Fd 2: July 25, 1978 memo on visit to Bell Northern by E&S, summarizing
      their proposed use of E&S products. Mentions Bell Northern's PDP-11/40 and
      /60, IBM 360 which will be replaced with a two Decsystem 20s, and a
      PDP-11/34 on order. No UNIX.
  - Fd 23–24: Brigham Young University, Correspondence and Agreement Proposal
    (1974-1987)
    - Fd 23: E&S gifted BYU a Picture System in 1979, which may have
      jump-started their Animation department.
    - Fd 24: E&S will repair BYU systems, if BYU allows E&S to use their system
      40hrs per 2 month period.
